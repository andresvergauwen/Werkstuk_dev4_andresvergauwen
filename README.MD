# Development 4: Opdracht Werkstuk

## Installation
What you need ? 
[NPM](https://www.npmjs.com/)

## Usage
Go to your terminal in the backend folder
run command
```bash
 npm install
```
then run 
```bash
nodemon server.js
```
 or 
```bash
npm start
```
to start the server.

after that
right click on index.html and click on open with live server.
